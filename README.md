AI-Based Feedback Collection & Notification System

Overview
This project is a simple yet effective feedback collection and communication system built using Google Forms, Google Sheets, and email automation concepts. It captures user responses, stores them in a structured dataset, and enables follow-up communication with participants.

Features
Collect user data via Google Form
Automatically store responses in Google Sheets
Email-based interaction with respondents


Capture important details like:
Name
Email ID
Phone Number

Organized dataset for analysis and insights
